# Grader AI

AI-assisted grading dashboard with OCR review and student scoring workflow, built with Vite + React + TypeScript.

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1UNNY9x2YI_lv5BricYqGPUnsQqXfzUNL

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
